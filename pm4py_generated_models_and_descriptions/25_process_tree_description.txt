process tree structure: ->( 'Prepare Order', +( 'Review Invoice', ->( 'Process Payment', *( 'Pack Item', 'Ship Package' ) ) ) )

Description: Delivery Service of e-commerce company

The process contains in total 5 activities from beginning to the end with three main parts: order preparation, order processing and shipment. When the order comes, the shop owner prepares the order. After gathering or checking customer information and payment details, the order can continue with thet process payment or its invoice needs to be reviewed. If the invoice is valid, the order is processed with payment. The item is packed afterward in box or bag and ship to the customer.
