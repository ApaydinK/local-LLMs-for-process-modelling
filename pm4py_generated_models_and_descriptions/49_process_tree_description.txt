Use Case: e-commerce

The shop gathers product information that owner sells, then handle product returns and simultanously update inventory. This is daily workflow and ends up at the end of the day or enough information to gather. In the selling branch where owner works with customers, he needs to confirm order details and ends up this order. Simultanously, he has to process order and after processing, the order need to be shipped and more information need to be added to inform with customer. This process ends up when all orders are processed.
